# RAVIAFEST

Sitio web promocional para **RAVIAFEST**, un festival musical de rap y trap. El proyecto está desarrollado en HTML, CSS y JavaScript y se centra en ofrecer una experiencia visual potente con animaciones, fuentes personalizadas y contenido multimedia.

## Arquitectura de la información
- `index.html` — Portada con hero animado, claim del festival, destacados y accesos directos.
- `lineup.html` — Cartel por días, horarios y fichas de artistas con recursos multimedia.
- `entradas.html` — Tipologías de abonos, tabla comparativa, FAQs y formulario de compra.
- `info.html` — Información logística: mapa del recinto, servicios, transporte y contacto.
- `styles.css` — Estilos globales: tipografía, layout responsivo y componentes reutilizables.
- `js/script.js` — Interacciones principales: menú móvil, animaciones y lógica del formulario.
